## 🌟 Overview
**ai-startup-factory** is a sophisticated multi-agent Python framework designed to simulate the startup ideation pipeline. AI agents collaborate to generate, critique, and refine startup concepts at scale.

## 🚀 Key Features
- 🤖 **Multi-Agent Architecture:** Specialized agents for ideation, evaluation, and refinement.
- 📊 **Data-Driven:** Configurable datasets and knowledge bases in data/.
- 🧪 **Tested:** Comprehensive test suite for agent reliability.
- 🔄 **CI/CD:** GitHub Actions pipeline for automated testing.

## 📦 Getting Started
`ash
