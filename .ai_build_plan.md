# ai-startup-factory - AI Build Blueprint

A Python-based AI agent system for simulating or automating startup ideation workflows.

## Goal
Maintain a robust multi-agent system for generating, evaluating, and refining startup ideas using AI pipelines.

## Tech Stack
- **Language:** Python.
- **AI Agents:** Custom agent framework (see gents/ directory).
- **Data:** Configuration and datasets in data/.
- **Testing:** Test suite in 	ests/.
- **CI:** GitHub Actions (.github/).

## AI Guidelines
1. All new agents must follow the existing agent interface pattern in gents/.
2. Tests in 	ests/ must be updated when adding new agent capabilities.
3. Never hardcode API keys — use environment variables and .env files (with .env.example as template).
4. Keep .ai_build_plan.md in the repository for architectural continuity.
